# Gracious Smiles

A premium custom WordPress website for a dental clinic, designed and developed with a modern UI/UX approach, responsive layouts, and a clean, trustworthy aesthetic.

## Project Overview

Gracious Smiles is a custom-built dental clinic website focused on:
- Modern and premium design
- Strong user experience
- Clear service presentation
- Mobile-first responsiveness
- Trust-building clinic branding
- Fast, clean, and maintainable code structure

## Features

- Custom WordPress theme
- Homepage with premium hero and service sections
- About Us page
- Services page
- Meet the Doctor page
- Contact page
- Responsive design for all devices
- Clean and reusable template parts
- Page-specific CSS files
- Optimized layout and typography
- Custom footer and header branding

## Tech Stack

- WordPress
- PHP
- HTML5
- CSS3
- JavaScript
- Custom WordPress theme architecture

## Setup

1. Download or clone the repository.
2. Place the theme folder inside your WordPress installation at:
   `wp-content/themes/`
3. Activate the theme from the WordPress dashboard.
4. Configure menus, pages, and theme settings as needed.

## Development Notes

- Each major page has its own template file.
- Page-specific styles are separated into dedicated CSS files.
- Template parts are used for better maintainability.
- The design follows a premium healthcare aesthetic with a modern and responsive layout.

## Images

All clinic and team images used in the project are stored in the theme assets folder and loaded where required.
